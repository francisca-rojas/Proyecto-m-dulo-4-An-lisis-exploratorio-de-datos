# Proyecto Módulo 4 
# Análisis Exploratorio de Datos (EDA)

##  Introducción

El presente proyecto corresponde al desarrollo de un **Análisis Exploratorio de Datos (EDA)** aplicado a un conjunto de datos que simula el comportamiento de clientes en un entorno de comercio digital, considerando dos canales de compra: Web y App.

El objetivo fue aplicar técnicas de preparación, análisis y visualización de datos utilizando Python, con el fin de identificar patrones, relaciones entre variables y generar conclusiones fundamentadas en evidencia cuantitativa.

---

##  Objetivos

### Objetivo General

Desarrollar un análisis exploratorio que permita comprender el comportamiento de los clientes y las diferencias entre los canales de compra.

### Objetivos Específicos

- Realizar limpieza y validación estructural del dataset.
- Analizar la distribución de variables .
- Comparar el comportamiento entre canal Web y App.
- Evaluar la relación entre variables cuantitativas.
- Generar visualizaciones interpretables.
- Elaborar conclusiones y recomendaciones basadas en datos.

---

##  Descripción del Dataset

El conjunto de datos incluye las siguientes variables:

- `Edad`
- `Genero`
- `Region`
- `Canal`
- `Visitas_Mensuales`
- `Compras`
- `Monto_Promedio`
- `Gasto_Total`
- `Devoluciones`
- `Satisfaccion`
